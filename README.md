# React-todo-list
This is a simple to-do list. User can enter What tasks he/she has to do, and can also delete it.

## Tech Stacks 
1. HTML 
2. CSS
3. Javascript
4. React

## Running the project
1. Fork the repo.
2. Clone the forked repo.
3. Open the project in VS Code or any terminal.
4. Type ``` npm start ``` in the terminal.
